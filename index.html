Get Started in 60 seconds

Easy oneliner install:

Windows (Powershell)
iwr https://deno.land/x/install/install.ps1 -useb | iex
Mac/Linux (Shell)
curl -fsSL https://deno.land/x/install/install.sh | sh
 

Test & show off remote URL executions:

deno run https://deno.land/std/examples/welcome.ts
More detailed instructions on Deno installation here: https://DenoCode.com/?page=start
including homebrew, choco, scoop & cargo install commands.

Example Code

server.tsjs

import { serve } from 'https://deno.land/std/http/server.ts';
const port = 8000;
const s = serve({ port });
console.log(`Listening on http://localhost:${port}/`);

for await (const req of s) {
	req.respond({ body: 'Hello Javascript!' });
}
